generate_wallet_spend_keys

  • generate_wallet_spend_keys(wallet_seed: string, subwallet_index?: number): Promise<[string, string]>

  • Generates deterministic subwallet keys from the wallet seed value.

    For compatibility & security purposes, all wallet keys and addresses, should be derived from the wallet seed and the seed should never be used directly.

    Subwallets are generated by injecting deterministic salt into the hashing process whereby it is nearly impossible to reconstruct the spend keys without knowing the subwallet index and the seed value that they were derived from.

    Parameters

    • wallet_seed: string
    • subwallet_index: number = 0

    Returns Promise<[string, string]>
